## Example Legal Services
A clean, responsive legal services landing page built to present an independent attorney with clarity and trust. It features a modern layout (Hero, About, Services, Why Choose Me, How It Works, Contact) and a professional, client-focused tone designed to convert visitors into consultation requests.

## Live Demo

https://example-legal-services.vercel.app/

## Features

Responsive layout (mobile-first)

Sections: Hero, About, Services, Why Choose Me, How It Works, Contact

Service cards (Family Law, Immigration, Contracts, Civil Disputes, Consultations)

Clear process timeline (3-step “How It Works”)

Contact section with phone + email and quick CTA prompts

Professional copy focused on clarity, confidence, and personal attention

## Tech Stack

Vite

React (TypeScript)

Tailwind CSS

lucide-react (icons)

## Run Locally
npm install
npm run dev
